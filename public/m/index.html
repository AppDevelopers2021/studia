Progressive Web App1

PWA 앱은 브라우저 캐싱을 통해 기기에 웹 앱을 설치하고 로딩 속도 및 성능을 향상시킬 수 있습니다.

소셜 로그인

번거로운 가입 절차 없이 구글 계정으로 간편하게 가입 및 로그인하세요

접근성

시각장애인, 스크린 리더 등도 페이지의 내용을 문제 없이 읽을 수 있도록 접근성 기술을 사용하였습니다.

다크 모드

다크 모드를 사용하여 눈의 피로도를 줄이고 배터리와 디스플레이도 보존하세요. 기기의 설정값이 그대로 반영됩니다.

오프라인 연결2

인터넷 연결이 불안정하거나 잠시 끊어진 경우에도 노트 작성이 가능합니다. 나중에 다시 네트워크에 연결되면 작업이 저장됩니다.3

DB 보안4

Firebase Realtime Database의 보안 규칙을 설정하여 본인이 아닌 경우에는 노트 접근을 차단하도록 했습니다.

  1. 프로그레시브 웹 앱 기술은 STUDIA 웹 앱에서만 사용 가능합니다. 또한 이를 사용하려면 Google Chrome 버전 39 이상, Microsoft Edge 버전 79 이상, Safari on iOS 버전 11.3 이상, Opera Mobile 버전 13 이상, Chrome for Android, Firefox for Android, 삼성 인터넷 또는 그 밖의 A2HS를 지원하는 브라우저를 사용해야 합니다. Microsoft Internet Explorer, Firefox Desktop, Safari on macOS, Opera Desktop 등의 브라우저에서는 PWA 기능을 사용할 수 없습니다.

  2. 오프라인 접근 기능은 네이티브 앱에서만 사용 가능합니다. 사용하려면 애플리케이션 설정에서 오프라인 접근 항목을 활성화하십시오.

  3. 데이터베이스 수정은 일시적 네트워크 끊김의 경우에만 사용 가능하며, 네트워크 연결이 완전히 끊어진 경우에는 읽기 모드만 지원됩니다. 노트, 메모 또는 과제 및 준비물을 오프라인 상태에서 생성하거나 수정, 삭제한 후 앱을 종료한 경우에는 해당 수정사항이 폐기됩니다.

  4. 데이터베이스에 저장되는 개인 데이터는 HTTPS 보호를 통해 전송되며 구글의 Firebase 데이터 센터(동아시아 서버)에서 AES 256-Bit로 암호화된 후 보관됩니다.
